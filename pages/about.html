this site

i tried to keep things as simple as possible. this means:

  • vanilla HTML, CSS, JS1
  • web server written in C2
  • art drawn in Paint

which in turn, means:

  • no dependencies or packages to deal with3
  • no complicated setup to run server4
  • no reliance on external platforms/providers5

and most importantly:

  • a fun project, and
  • a good learning experience!

and that's all there is to it. i mostly wanted to do something different instead of reaching for the same tools i've been using forever (node + front-end framework).

if you're interested in more details, feel free to view the github repo.

if you have any questions, concerns, or something you want to share, email me at kevin@kevinjlee.xyz
